# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly). It allows users to create a secure account where they can freely log, add, edit or delete the shortened URLs of their favorite websites.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
